# Programación Web - Ejercicios de HTML, CSS y JavaScript

Repositorio con los ejercicios prácticos de la materia **Programación Web**, del Instituto Tecnológico de Oaxaca, Departamento de Sistemas y Computación.

## Descripción de la actividad

En estos 10 ejercicios prácticos de programación web, me basé en ver cuáles son los fundamentos y elementos de HTML, CSS3 y JavaScript, aplicando diferentes conceptos en cada práctica.

En el primer ejercicio fue crear una página HTML básica con la estructura mínima (`<!DOCTYPE html>`, `<head>`, `<body>`). Fui avanzando etiqueta por etiqueta: formato de texto (`<strong>`, `<em>`), enlaces (`<a>`), imágenes (`<img>`), secciones semánticas (`<header>`, `<section>`, `<footer>`), listas, tablas con `<thead>`/`<tbody>` para un horario de clases, y un formulario de contacto completo con validación HTML y estilizado en CSS externo usando variables (`:root`).

En el último ejercicio (10), tenía que hacer una calculadora de conversión de temperatura (Celsius a Fahrenheit), en el cual se ocupa el concepto del **DOM** (Document Object Model): JavaScript lee lo que el usuario escribe, asegura que el dato sea correcto, hace el cálculo y muestra el resultado en pantalla, usando eventos como `submit` y métodos como `preventDefault()`. Para este último ejercicio me basé en lo aplicado en los demás ejercicios y buscando los elementos en los enlaces de Classroom
